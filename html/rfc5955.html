Abstract

摘要

This document defines a new media type for TimeStampedData envelopes as described in RFC 5544.

1. Introduction
1. 介绍

This document defines a new media type for TimeStampedData envelopes. A TimeStampedData envelope, described in [RFC5544], binds a file with one or more time-stamp tokens obtained for that file. A media type registration, lacking in [RFC5544], enhances shareability.

本文档为TimestamedData信封定义了一种新的媒体类型。[RFC5544]中描述的TimeStampedData信封将文件与为该文件获得的一个或多个时间戳标记绑定。[RFC5544]中缺少的媒体类型注册增强了可共享性。

Applications which use this media type:

使用此媒体类型的应用程序：

Any application that exchanges TimeStampedData envelopes over a MIME-based transport and possibly processes them, either directly or via external handlers or viewers. Such applications may be, for instance, digital signature applications, archival systems, email clients, web browsers, etc.

通过基于MIME的传输交换TimestamedData信封并可能直接或通过外部处理程序或查看器对其进行处理的任何应用程序。例如，此类应用程序可以是数字签名应用程序、档案系统、电子邮件客户端、web浏览器等。
