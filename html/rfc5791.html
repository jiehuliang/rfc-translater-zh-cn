RFC 2731 ("Encoding Dublin Core Metadata in HTML") Is Obsolete

RFC2731（“用HTML编码都柏林核心元数据”）已经过时

Abstract

摘要

This document obsoletes RFC 2731, "Encoding Dublin Core Metadata in HTML", as further development of this specification has moved to the Dublin Core Metadata Initiative (DCMI).

由于本规范的进一步开发已转移到都柏林核心元数据倡议（DCMI），本文件废除了RFC 2731“用HTML编码都柏林核心元数据”。

1. Introduction
1. 介绍

[RFC2731] defines "Encoding Dublin Core Metadata in HTML". Newer specifications published by the Dublin Core Metadata Initiative (http://dublincore.org/) over the last decade, in particular "Expressing Dublin Core metadata using HTML/XHTML meta and link elements" ([DC-HTML]), have obsoleted this work.

[RFC2731]定义了“用HTML编码都柏林核心元数据”。都柏林核心元数据计划发布的更新规范(http://dublincore.org/)在过去十年中，特别是“使用HTML/XHTML元和链接元素表达都柏林核心元数据”（[DC-HTML]），已经淘汰了这项工作。

Note: Implementers of RFC 2731 should note that DC-HTML introduces a new requirement with respect to the use of the "head" element's "profile" attribute.

注意：RFC2731的实现者应该注意，DC-HTML引入了一个关于使用“head”元素的“profile”属性的新需求。
